# Imagem com Link

Este é um projeto simples de HTML e CSS que demonstra como utilizar imagens como links clicáveis. Idealizado como parte da minha prática no curso de Desenvolvimento Web Full Stack, o objetivo principal é exercitar o uso correto de elementos visuais, navegação e estrutura de uma página.

## 🔗 Acesse o Projeto

Você pode visualizar o site publicado aqui:  
👉 [Clique para abrir](https://jaqueline-gotardi.github.io/imagem-com-link/)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

## 💡 Funcionalidades

- Exibição de imagens organizadas em blocos.
- Cada imagem funciona como um link externo (abre em nova aba).
- Estilo visual básico aplicado com CSS.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como um exercício prático para reforçar conhecimentos iniciais de HTML e CSS, especialmente sobre:

- Estruturação semântica de páginas;
- Inserção de imagens;
- Criação de links com `target="_blank"`;
- Uso básico de estilos CSS para posicionamento e apresentação.

## 📈 Possíveis melhorias futuras

- Tornar o layout responsivo para dispositivos móveis;
- Adicionar um título e descrição para cada imagem;
- Incluir animações ao passar o mouse;
- Melhorar a organização e reutilização do CSS.
